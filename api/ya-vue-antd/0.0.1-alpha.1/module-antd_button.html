    Button（按钮）

    module:antd/button

    Props

    Name Type Default value Required? Description
    disabled boolean false No 按钮失效状态
    ghost boolean false No 幽灵属性，使按钮背景透明
    href string - No 点击跳转的地址，指定此属性 button 的行为和 a 链接一致
    htmlType string button No 设置 button 原生的 type 值，可选值请参考 HTML 标准
    icon string - No 设置按钮的图标类型
    loading boolean|object false No 设置按钮载入状态
    loading.delay number - No 设置按钮载入延时消失时间(ms)
    shape string - No 设置按钮形状，可选值为 circle 或者不设
    size string default No 设置按钮大小，可选值为 small large 或者不设
    target string - No 相当于 a 链接的 target 属性，href 存在时生效
    type string - No 设置按钮类型，可选值为 primary dashed danger或者不设
    block boolean false No 将按钮宽度调整为其父宽度的选项
    onClick onClick (event) => void No 点击按钮时的回调，onClick，查看示例
